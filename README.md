Building
--------
 1. Clone the project and cd in it.
 2. Run:
 
    ```
    cabal sandbox init
    cabal install --only-dependencies
    cabal build
    ```
    
 3. Built binary will reside in the `dist/build/Scarybot` subdirectory.
